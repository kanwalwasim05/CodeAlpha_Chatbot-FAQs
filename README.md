# 🤖 FAQ Chatbot - CodeAlpha Task 2

## 📌 Project Overview
An intelligent FAQ chatbot built for CodeAlpha AI Internship that uses Natural Language Processing (NLP) to answer questions about AI, Machine Learning, and CodeAlpha.

## ✨ Features
- 🤖 Smart question matching using cosine similarity
- 📚 15+ pre-built FAQs about AI/ML topics
- 🔍 NLP preprocessing (tokenization, stopword removal)
- 💬 Beautiful, responsive chat UI
- 🎯 Quick suggestion chips
- ⌨️ Keyboard shortcuts (Enter to send)
- 📱 Mobile-friendly design

## 🛠️ Tech Stack
- HTML5, CSS3, JavaScript
- Custom NLP implementation (no external libraries)
- Cosine similarity algorithm
- Web Speech API (optional - can be added)

## 🚀 Quick Start
1. Download `index.html`
2. Double-click to open in browser
3. Start chatting!

## 📖 How It Works
1. **User Input**: User types a question
2. **Preprocessing**: Text is tokenized and cleaned
3. **Matching**: Cosine similarity compares with FAQs
4. **Response**: Best matching answer is displayed
